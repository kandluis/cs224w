    \subsection{1. Analyzing the Wikipedia voters network {[}9
points{]}}\label{analyzing-the-wikipedia-voters-network-9-points}

Download the Wikipedia voting network wiki-Vote.txt.gz:
http://snap.stanford.edu/ data/wiki-Vote.html.

Using one of the network analysis tools above, load the Wikipedia voting
network.

Note that Wikipedia is a directed network. Formally, we consider the
Wikipedia network as a directed graph \(G = (V, E)\), with node set
\(V\) and edge set \(E ⊂ V × V\) where (edges are ordered pairs of
nodes). An edge \((a, b) ∈ E\) means that user \(a\) voted on user
\(b\).

To make our questions clearer, we will use the following small graph as
a running example: \[
Gsmall = (Vsmall, Esmall)
\] where \[
Vsmall = \{1, 2, 3\}
\] and \[
Esmall = \{(1, 2),(2, 1),(1, 3),(1, 1)\}
\]
